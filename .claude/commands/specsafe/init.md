---
name: "SPECSAFE: Init"
description: Initialize SpecSafe project
---

Initialize a new SpecSafe project.

**Usage:** `/specsafe:init [project-name]`

This will create:
- specs/active/, specs/completed/, specs/archive/
- specsafe.config.json
- PROJECT_STATE.md

**Next:** Create first spec with `/specsafe:new`
