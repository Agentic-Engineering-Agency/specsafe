# SpecSafe Rules for Continue.dev

## Version: 1.0.0

This is a placeholder. Full content will be added in a future update.

## Overview

Continue.dev integration for SpecSafe projects provides custom commands and context providers
for working with specs directly from VS Code.

## Commands

- `/specsafe` - Show current project status
- `/spec` - Show details for a specific spec

## Context Providers

- `specsafe-state` - Automatically includes PROJECT_STATE.md in context
